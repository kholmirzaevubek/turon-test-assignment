# Laravel 11 Project

This project is built using Laravel 11 and Tailwind CSS.

## Requirements

- PHP 8.2 or higher
- Composer
- Node.js and NPM
- Laravel 11

## Installation

1. **Install composer**:
```bash
    composer install
```
2. **Create an .env file**:
```bash
    cp .env.example .env
```
3.**Create the database**:
```bash
    php artisan migrate
```

4.**Run the seeder**:
```bash
    php artisan db:seed
```

5.**Run storage link**:
```bash
    php artisan stroge:link
```

6.**Install NPM dependencies**:
```bash
    npm install
```
    
7.**Build Tailwind CSS**:
```bash
    npm run build
```
    
8.**Run the project**:
```bash
    php artisan serve
```
